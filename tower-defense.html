Application Tower Defense

Simulation of game Tower defense. Programming language used: C++ (Qt5). Source code can be found:

On start screen of the application, there are buttons where player can choose level, read about and quit a game.

In about, player can read some instructions how to play a game. Information about levels, enemies and towers. Each enemy unit has fields of health, speed, and type(is it flying unit, or ground) and how much money you will earn by killing it. Each tower has fields about damage dealt to enemy, price and type (what kind of units it can attack). All that information are included in about.

After reading about section, players can start playing a game. User should choose which level they want to play. After choosing, game starts, and now fun part comes. Below is video how does it look like:

On left side, there are towers which you can take by dragging them with the mouse and putting then on green area. On bottom bar, there are information about score, number of lives left, and left gold which you can spent. On right bottom corner there are two buttons, to mute music and to quit a game. Enemies start spawning when player clicks button attack.
I hope you will enjoy playing this game, specially while playing level 3, which was a big challenge also for me.
